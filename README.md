# Bath Sommelier

Bath Sommelier는 사용자의 오늘 컨디션과 목욕 환경을 바탕으로, 지금 가장 적합한 홈 배스 루틴을 추천해주는 프리미엄 웰니스 앱입니다.

<p align="center">
  <img src="assets/readme/home.png" alt="Bath Sommelier home screen" width="280" />
</p>

## Overview

- 짧은 입력만으로 오늘 상태에 맞는 맞춤형 배스 루틴 추천
- 추천 이유, 진행 방법, 안전 안내를 함께 제공하는 설명형 경험
- 준비, 진행, 마무리, 회고까지 이어지는 루틴 중심 플로우
- 홈 스파와 부티크 호텔 배스 경험을 닮은 조용한 프리미엄 톤
