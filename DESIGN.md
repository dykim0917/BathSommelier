# Design System — 바스타임

## Product Context
- **What this is:** 그냥 씻고 끝내던 시간을, 오늘 몸 상태에 맞는 샤워·목욕 루틴으로 바꿔주는 모바일 셀프케어 앱.
- **One-line definition:** 피곤한 날, 잠 안 오는 밤, 운동 후 뻐근한 몸까지 오늘 몸에 맞게 씻는 방법을 가볍게 안내하는 앱.
- **Who it's for:** 매일 샤워는 하지만, 피곤하거나 몸이 무거운 날에는 씻는 시간을 조금 더 잘 활용하고 싶은 20-40대 사용자.
- **Space/industry:** 데일리 셀프케어, 컨디션 기반 샤워 루틴, 안전한 목욕·샤워 가이드.
- **Project type:** 모바일 앱 중심의 컨디션별 샤워·목욕 루틴 가이드.

## Brand Messaging Principle

### Core Message

그냥 씻고 끝내던 시간을, 오늘 몸에 맞는 루틴으로 바꿔준다.

### Core User Moment

사용자는 보통 "오늘 어떻게 씻지?"라고 고민하지 않는다.
대부분 그냥 씻는다.

바스타임은 이미 존재하는 고민을 해결하기보다,
매일 하는 씻기 행동에 새로운 의미와 방법을 제안한다.

사용자가 떠올려야 하는 장면은 다음과 같다.

> 오늘 좀 피곤하고 찌뿌둥한데, 그때 봤던 것처럼 그렇게 씻어볼까?

### Hero Copy

그냥 씻기 아쉬운 날에

### Supporting Copy

피곤한 날, 잠 안 오는 밤, 운동 후 뻐근한 몸까지.
오늘 몸에 맞는 샤워·목욕 루틴을 가볍게 따라가요.

## Aesthetic Direction
- **Direction:** Everyday Wash Ritual
- **Decoration level:** Intentional
- **Mood:** 편안하지만 들뜨지 않고, 전문적이지만 과시하지 않는다. 사용자는 화면을 볼 때 "오늘은 그냥 씻지 말고 이렇게 가볍게 해볼까"를 바로 떠올릴 수 있어야 한다.
- **Reference cues:** 생활형 셀프케어, 샤워 루틴, 안전한 목욕 가이드, 차분한 기록 앱, 물과 증기의 깨끗한 감각.
- **Core feeling:** 호텔 스파처럼 특별한 시간이 아니라, 집에서 매일 하는 씻는 시간을 오늘 몸에 맞게 살짝 바꿔주는 느낌.

## Logo System
- **Primary mark:** `assets/images/bathtime.svg`
- **Concept:** 물방울, 물의 흐름, 루틴의 순서를 하나의 심볼로 묶는다.
- **Usage:** 앱 헤더, 온보딩 완료, 앱 아이콘, 스플래시, 랜딩 첫 브랜드 신호에 사용한다.
- **Rule:** 작은 크기에서는 심볼만 사용하고, 설명 문구로 기능을 보완한다.

## Typography
- **Display/Hero:** System sans
  브랜드 헤드라인, 루틴 타이틀, 주요 섹션 제목도 산세리프로 유지한다.
- **Body/UI:** System sans / `sans-serif`
  설명, 버튼, 폼, 보조 카피는 모두 산세리프로 고정한다.
- **Data/Tables:** monospace fallback
  타이머, 수치, 진행 시간, 기록 데이터는 고정폭 계열을 쓴다.
- **Code:** `SpaceMono`
- **Scale:** Hero 34/40, section 20/26, card 18/24, body 14-15/20-23, caption 12/17-18

## Color
- **Approach:** Restrained
- **Primary background:** `#102629`
- **Top gradient:** `#193A3E`
- **Bottom gradient:** `#07191B`
- **Surface:** `rgba(22, 45, 48, 0.9)`
- **Surface soft:** `rgba(18, 39, 42, 0.76)`
- **Primary text:** `#F7F3EA`
- **Secondary text:** `#D7E1DC`
- **Muted text:** `#9FB5AF`
- **Accent / water:** `#94D2BF`
- **Accent soft:** `rgba(148, 210, 191, 0.16)`
- **Soft border:** `rgba(230, 246, 239, 0.14)`
- **Strong border:** `rgba(148, 210, 191, 0.42)`
- **Semantic:** warning `#CAA071`, danger `#C28676`
- **Dark mode strategy:** 이 앱은 dark-steam baseline 하나로 운영한다. 별도 라이트 모드를 억지로 확장하지 않는다.

## Spacing
- **Base unit:** 4px
- **Density:** Comfortable
- **Scale:** 4, 8, 12, 16, 20, 24, 32, 40
- **Rule:** 버튼과 카드 내부 패딩은 기존보다 약간 넓게 가져간다.

## Layout
- **Approach:** Hybrid
- **App shell:** 정돈된 단일 컬럼과 안정적인 루틴 카드 스택.
- **Editorial moments:** 홈 히어로, 트립 카드, 바스타임 완료 화면에서만 에디토리얼 톤을 허용한다.
- **Border radius:** card 12, large card 16, button 14, pill full.
- **Grid:** 모바일 우선. 좌우 패딩 20-24를 기본으로 유지한다.

## Motion
- **Approach:** Minimal-functional
- **Use:** 페이드, 부드러운 등장, 진행 상태 피드백.
- **Avoid:** 과한 bounce, 쓸모없는 장식 전환, 의미 없는 패럴랙스.
- **Rule:** 물, 증기, 타이머처럼 제품의 본질과 닿는 곳에서만 감각적 모션을 사용한다.

## Voice In UI
- **Tone:** 생활형 셀프케어, 안전한 안내, 바로 실행할 수 있는 문장.
- **Messaging center:** 사용자가 씻는 법을 모른다고 말하지 않는다. 대신 그냥 씻던 시간을 오늘 몸에 맞게 조금 바꿔보자고 제안한다.

### Do

- "그냥 씻기 아쉬운 날에"
- "오늘 몸에 맞는 루틴"
- "온도와 시간만 확인해요"
- "필요한 것만 보고 바로 시작해요"
- "샤워만으로도 충분해요"
- "무리하지 않고 가볍게 따라가요"
- "씻는 동안은 타이머만 따라오세요"

### Don't

- "씻는 법을 알려드릴게요"처럼 사용자가 모른다는 전제로 말하지 않는다.
- 목욕/입욕만 강조하지 않는다.
- 몽환적 추상 카피를 쓰지 않는다.
- 과도한 힐링, 치유, 치료 표현을 쓰지 않는다.
- 자기관리 압박처럼 말하지 않는다.
- 전문가처럼 훈계하지 않는다.
- 루틴 완료를 성취처럼 몰아가지 않는다.

## Component Rules
- 첫 화면과 주요 CTA는 "씻는 시간을 가볍게 바꾸는 경험"을 먼저 보여준다.
- 루틴 상세는 긴 설명보다 3초 요약을 우선한다.
- 추천 이유, 준비물, 제품 추천은 시작을 막지 않고 필요할 때만 펼쳐볼 수 있게 한다.
- 샤워, 족욕, 욕조 중 샤워를 가장 낮은 진입장벽으로 보여준다.
- 욕조 이미지는 감성 확장으로 사용하되, 앱이 욕조 전용처럼 보이지 않게 한다.
- 타이머 화면은 사용자가 씻는 동안 읽지 않아도 따라갈 수 있도록 단순하게 유지한다.
- 완료 화면은 성취보다 "오늘의 방식과 시간을 차분히 남기는 기록"으로 보여준다.
