## Safety Override
- If user has `hypertension_heart`, recommended/max temperature is hard-capped to `38°C`.
- This applies to both Care and Trip recommendations.

## Core UX Changes
- Home screen has top tabs: `Care` / `Trip`.
- Common environment toggle at bottom: `욕조/족욕/샤워`.
- Last selected environment is persisted and restored.
- Completion feedback copy supports theme insertion: `오늘의 [테마] 어떠셨나요?`.
