# 바스타임 전체 화면 역와이어프레임

작성일: 2026-04-27  
기준: 현재 구현 코드 + Codex in-app browser `http://localhost:8081` 확인  
목적: 온보딩부터 결과 화면까지, 사용자가 실제로 보는 문구 기준으로 전체 화면 구조를 한 문서에 정리한다.

## 공통 하단 탭

대상: 온보딩/결과/법적 문서 외 메인 탭 화면

```text
┌────────────────────────────┐
│ 홈  컨디션  무드  제품  프로필 │
└────────────────────────────┘
```

탭 라벨:

- `홈`
- `컨디션`
- `무드`
- `제품`
- `프로필`

## 7. 컨디션 탭

경로: `/(tabs)/care`

```text
┌────────────────────────────┐
│ 컨디션 가이드              │
│ 컨디션 루틴                │
│ 지금 몸 상태에 맞춰 무리    │
│ 없는 루틴을 골라보세요.     │
│                            │
│ 가능한 방식                 │
│ [샤워] [족욕] [욕조]        │
│                            │
│ 컨디션별 루틴              │
│ 지금 컨디션에 맞는 추천을   │
│ 바로 시작하고, 필요할 때만  │
│ 세부 느낌을 더 골라보세요.  │
│                            │
│ ┌────────────────────────┐ │
│ │ 먼저 보면 좋아요        │ │
│ │ 운동 후 뻐근함을 풀어볼까요? │
│ │ {현재 방식 기준 설명}   │ │
│ │ {방식 적합도} {안전 배지} │
│ │ 간단히 시작 ›           │ │
│ │ 다르게 고르기           │ │
│ └────────────────────────┘ │
│ ┌────────────────────────┐ │
│ │ 빠르게 시작하기         │ │
│ │ 잠들기 어려울 때 좋은 루틴 │
│ │ 간단히 시작 ›           │ │
│ │ 다르게 고르기           │ │
│ └────────────────────────┘ │
│ ...                        │
└────────────────────────────┘
```

컨디션 카드 목록:

- `운동 후 뻐근함을 풀어볼까요?`
- `잠들기 어려울 때 좋은 루틴`
- `술 마신 다음엔 이렇게 시작해보세요`
- `붓기가 느껴질 때 해보세요`
- `감기 기운이 느껴질 때`
- `생리통이 있을 때`
- `스트레스를 풀고 싶을 때`
- `기분 전환이 필요할 때`

### 컨디션 세부 선택 모달

진입: 컨디션 카드의 `다르게 고르기`

```text
┌────────────────────────────┐
│ {선택한 컨디션 카드 제목}   │
│                            │
│ ┌────────────────────────┐ │
│ │ {세부 선택 라벨}        │ │
│ │ {세부 선택 설명}        │ │
│ └────────────────────────┘ │
│ ┌────────────────────────┐ │
│ │ {세부 선택 라벨}        │ │
│ │ {세부 선택 설명}        │ │
│ └────────────────────────┘ │
│                            │
│ [닫기]                      │
└────────────────────────────┘
```

대표 세부 선택 문구:

- `하체가 뻐근해요` / `하체 중심으로 부드럽게 풀어줘요.`
- `어깨/목이 뻐근해요` / `상체 이완에 더 집중해요.`
- `전신이 무거워요` / `전신을 고르게 풀어주는 기본 루틴이에요.`
- `생각이 많아요` / `호흡 구간으로 머리를 천천히 가라앉혀요.`
- `예민해서 잠이 안 와요` / `자극을 줄이고 안정적으로 진행해요.`

```text
┌────────────────────────────┐
│ 입문자 추천                │

│ 먼저 골라봤어요.            │
│                            │
│ 카테고리                   │
│ [전체] [에센셜오일] [바스솔트] │
│ [목욕 아이템] [바디워시] [허브] │
│                            │
│ 바로 보기 좋은 제품         │
│ {n}개 제품 · 입문자가 바로   │
│ 써보기 쉬운 완제품만 모았어요 │
│                            │
│ ┌────────────────────────┐ │
│ │ {제품 카드}             │ │
│ │ {브랜드/제품명}         │ │
│ │ {설명}                  │ │
│ │ 제품 보기               │ │
│ └────────────────────────┘ │
└────────────────────────────┘
```

### 제품 상세 모달

```text
┌────────────────────────────┐
│ {제품명}                    │
│ {브랜드}                    │
│ {설명}                      │
│                            │
│ [판매처 보기]               │
│ [제품 목록으로 돌아가기]    │
└────────────────────────────┘
```

판매처가 없거나 링크 열기에 실패하면:

- `판매처 링크 없음`
- `아직 연결된 판매처 링크가 없어요.`
- `링크 열기 실패`
- `지금은 판매처 링크를 열 수 없어요.`

## 10. 프로필 탭 - 기록

경로: `/(tabs)/my` 또는 `/(tabs)/history`

```text
┌────────────────────────────┐
│ 프로필                     │
│ 기록을 돌아보고, 내 환경과  │
│ 상태를 한곳에서 차분히      │
│ 관리하세요.                 │
│                            │
│ [기록] [설정]              │
│                            │
│ ┌ 이번주 바스타임 ───────┐ │
│ │ 오늘의 바스타임 완료/전 │ │
│ │ Mon Tue Wed Thu Fri... │ │
│ │ 이번주 {n} / 7일        │ │
│ │ {n}일 연속 · {n}주 연속 │ │
│ └────────────────────────┘ │
│                            │
│ ┌ 이번 달 요약 ─────────┐ │
│ │ {n}회 완료             │ │
│ │ 평균 {n}분 · {방식}    │ │
│ │ 자주 찾은 테마: {테마} · {가중치} │
│ └────────────────────────┘ │
│                            │
│ [전체] [컨디션] [무드]      │
│                            │
│ {n}개의 루틴                │
│ ┌──────────┐ ┌──────────┐ │
│ │ {모드}   │ │ {모드}   │ │
│ │ {제목}   │ │ {제목}   │ │
│ │ {온도}°C · {입욕유형}  │ │
│ │ {월/일}  │ │ {월/일}  │ │
│ └──────────┘ └──────────┘ │
└────────────────────────────┘
```

기록이 없을 때:

```text
┌────────────────────────────┐
│ 첫 루틴을 시작해보세요      │
│ 아직 기록이 없어요          │
│ 첫 루틴을 시작하면 여기에    │
```

## 17. 현재 UX 원칙 요약

- 첫 선택은 `샤워 → 족욕 → 욕조` 순서로 보여준다.
- 카드에서 `간단히 시작`은 세부 모달 없이 레시피의 `3초 요약`으로 이동한다.
- 레시피 상세에서 실제 시작 CTA는 `시작하기`다.
- 추천 이유, 준비물, 제품 추천은 `자세히 보기` 아래로 접어 시작을 막지 않는다.
- 안전 안내는 시작 전 확인 가능한 위치에 유지한다.
- 완료 피드백은 `좋았어요 / 아쉬웠어요`로 선택형 보조 액션처럼 보여준다.
