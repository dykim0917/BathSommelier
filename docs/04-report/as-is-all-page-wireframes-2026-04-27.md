상태: 현재 브라우저 로컬 저장소에는 욕조 환경, 오늘 1회 완료 기록, `rec_1777251357136_d1digs` 기록이 존재함

```text

│ 홈  컨디션  무드  제품  프로필 │
```

## 17. 현재 문서화에서 제외/주의한 상태

- `/history`, `/settings` 직접 경로는 현재 탭 레이아웃만 노출되어 실사용 화면은 `/my`와 `/my?tab=settings` 기준으로 정리했다.
- 상세 화면의 수치/제품명/시간은 현재 선택된 `rec_1777251357136_d1digs` 기준 예시다.
- 제품 탭은 현재 전체 카테고리 기준 9개 제품이 모두 DOM에 노출된 상태를 기준으로 정리했다.
- 법적 문서는 화면 전문이 긴 스크롤로 노출된다. 본 문서에는 IA를 확인하기 위한 주요 조항 제목과 대표 본문을 유지했다.
