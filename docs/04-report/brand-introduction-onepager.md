# Bath Sommelier 브랜드 소개서

## 1. 프로젝트 한 줄 소개
Bath Sommelier는 사용자의 오늘 컨디션과 목욕 환경을 바탕으로, 지금 가장 적합한 홈 배스 루틴을 추천해주는 프리미엄 웰니스 앱입니다.

## 2. 우리가 풀고 있는 문제
많은 사용자가 피로, 스트레스, 수면 문제, 기분 저하를 느끼지만 정작 "오늘 내 상태에 어떤 목욕이 맞는지"는 잘 모릅니다.  
Bath Sommelier는 이 막연함을 줄이고, 사용자가 짧은 입력만으로도 안전하고 설득력 있는 맞춤형 루틴을 바로 시작할 수 있도록 돕습니다.

## 3. 제품이 제공하는 핵심 가치
- 내 상태를 이해하고 맞춤형 목욕을 설계해준다.
- 추천 이유를 설명해 사용자가 납득하고 실행하게 만든다.
- 추천에서 끝나지 않고 준비, 진행, 마무리까지 루틴 경험으로 연결한다.
- 기록과 회고를 통해 점점 더 나에게 맞는 웰니스 습관을 만든다.

## 4. 타깃 사용자
- 20-40대 웰니스 관심층
- 홈 스파, 향, 입욕제, 부티크 호텔 배스 경험에 관심 있는 사용자
- 과장된 힐링 메시지보다 실용적이면서도 고급스러운 가이드를 선호하는 사용자
- 초보자이지만, 자신에게 맞는 루틴을 믿고 따라가고 싶은 사용자

## 5. 현재 브랜드 방향
Bath Sommelier의 현재 방향성은 **"Quiet Luxury Spa"**입니다.  
명상 앱처럼 추상적이기보다는, 조용하고 신뢰감 있는 배스 코치에 가깝습니다.  
브랜드는 프리미엄이어야 하지만 과시적이지 않아야 하며, 감성적이어야 하지만 언제나 다음 행동이 분명해야 합니다.

## 6. 브랜드가 전달해야 하는 인상
- 조용한 프리미엄
- 신뢰 가능한 웰니스 가이드
- 집에서도 누릴 수 있는 호텔급 배스 경험
- 감각적이지만 과장되지 않은 절제
- 안전과 디테일을 챙기는 브랜드

## 7. 현재 제품 무드 키워드
Home Spa / Boutique Hotel Amenities / Deep Steam / Warm Ivory / Brass Accent / Editorial Wellness / Calm Coaching

## 8. 브랜딩 파트너에게 기대하는 것
- Bath Sommelier만의 고유한 브랜드 스토리와 언어 정리
- 로고, 심볼, 컬러, 타이포그래피를 포함한 일관된 브랜드 시스템 제안
- "프리미엄 웰니스"와 "실용적인 추천 서비스"가 함께 느껴지는 톤앤매너 설계
- 앱 UI, 랜딩 페이지, 패키지/콘텐츠 확장까지 고려한 활용성 있는 브랜드 가이드

## 9. 한 문장으로 정리하면
Bath Sommelier는 **"오늘의 나에게 가장 잘 맞는 목욕을 설계해주는, 조용하고 고급스러운 웰니스 가이드"**를 만들고자 합니다.
