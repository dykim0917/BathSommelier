## 1. 프로젝트 한 줄 소개
배쓰타임은 오늘 상태와 환경에 맞춰 무리 없이 따라할 목욕·샤워 루틴을 안내하는 생활형 셀프케어 앱입니다.

## 2. 우리가 풀고 있는 문제
많은 사용자가 피로, 스트레스, 수면 문제, 기분 저하를 느끼지만 정작 "오늘 내 상태에 어떤 목욕이 맞는지"는 잘 모릅니다.  
배쓰타임은 이 막연함을 줄이고, 사용자가 짧은 입력만으로도 안전하고 설득력 있는 맞춤형 루틴을 바로 시작할 수 있도록 돕습니다.

## 3. 제품이 제공하는 핵심 가치
- 내 상태를 이해하고 맞춤형 목욕을 설계해준다.
- 추천 이유를 설명해 사용자가 납득하고 실행하게 만든다.
- 추천에서 끝나지 않고 준비, 진행, 마무리까지 루틴 경험으로 연결한다.
- 기록과 회고를 통해 점점 더 나에게 맞는 쉬는 방식을 만든다.

## 4. 타깃 사용자
- 20-40대 생활형 셀프케어 관심층
- 목욕, 샤워, 족욕을 더 잘 활용하고 싶은 사용자
- 과장된 힐링 메시지보다 실용적이고 안전한 가이드를 선호하는 사용자
- 초보자이지만, 자신에게 맞는 루틴을 믿고 따라가고 싶은 사용자

## 5. 현재 브랜드 방향
배쓰타임의 현재 방향성은 **"Everyday Bath Guide"**입니다.  
명상 앱처럼 추상적이기보다는, 오늘 집에서 바로 따라할 수 있는 목욕·샤워 가이드에 가깝습니다.  
브랜드는 편안해야 하지만 흐릿하지 않아야 하며, 안전을 말하되 의료 서비스처럼 딱딱해지지 않아야 합니다.

## 6. 브랜드가 전달해야 하는 인상
- 편안한 생활형 셀프케어
- 신뢰 가능한 목욕·샤워 가이드
- 집에 있는 방식으로 바로 시작할 수 있는 현실감
- 감각적이지만 과장되지 않은 절제
- 안전과 디테일을 챙기는 브랜드

## 7. 현재 제품 무드 키워드
Daily Self-care / Clean Steam / Water Accent / Warm Ivory / Safe Guide / Calm Coaching

## 8. 브랜딩 파트너에게 기대하는 것
- 배쓰타임만의 고유한 브랜드 스토리와 언어 정리
- 로고, 심볼, 컬러, 타이포그래피를 포함한 일관된 브랜드 시스템 제안
- "생활형 셀프케어"와 "안전한 추천 서비스"가 함께 느껴지는 톤앤매너 설계
- 앱 UI, 랜딩 페이지, 패키지/콘텐츠 확장까지 고려한 활용성 있는 브랜드 가이드

## 9. 한 문장으로 정리하면
