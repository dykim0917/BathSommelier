# Home/My Wireframe (Streak 반영)

## Home 탭

### 목적
- 오늘 추천 루틴 진입
- 이번주 루틴 진행 현황 확인
- 주간 페이스 유지 동기 제공

### 레이아웃 (상→하)
1. 헤더
- 동적 타이틀
- 보조 문구
- 초보자 가이드(기록 0건일 때)

2. 이번주 루틴 카드 (신규 핵심)
- 타이틀: `이번주 루틴`
- 상태: `오늘 Bath 완료 ✔` 또는 `오늘 Bath 미완료 -`
- 주간 체크: `Mon Tue Wed Thu Fri Sat Sun` + `✔/-`
- 주간 카운트: `X / 7 baths this week`
- 스트릭: `N day streak · M weeks active`

3. 환경 칩
- 욕조 / 부분입욕 / 샤워

4. 루틴 섹션 2개
- 케어 루틴(프리뷰 4개)
- 트립 루틴(프리뷰 4개)
- 각 섹션 더보기

5. 최근 루틴
- 가로 스크롤 카드
- 없으면 빈 상태 카드

6. 디스클로저
- 안전 고지

### 상태/로직
- Daily Check: 월~일 기준 주간 7칸
- Weekly Bath Count: 이번 주 완료 횟수 / 7
- Daily Streak: 오늘부터 연속 완료 일수
- Weekly Streak: 주 1회 이상 완료한 연속 주차

## My 탭 > 기록

### 목적
- 누적 기록 조회
- 필터(전체/케어/트립)
- 동일한 주간 루틴 맥락 제공

### 레이아웃 (상→하)
1. 페이지 타이틀/서브타이틀

2. 이번주 루틴 카드 (Home과 동일 구성)
- `오늘 Bath 완료/미완료`
- `Mon~Sun 체크`
- `X / 7 baths this week`
- `N day streak · M weeks active`

3. 월간 요약 배너
- 총 완료 횟수
- 평균 시간/자주 사용 환경
- 자주 찾은 테마

4. 필터 칩
- 전체 / 케어 / 트립

5. 기록 카드 그리드
- 2열 카드
- 카드: 제목/수온/입욕 타입/날짜
- 탭 시 레시피 상세 이동

### 상태/로직
- streak 집계는 completionSnapshot.completedAt 기반
- Home과 동일 집계 함수 사용(월~일, 주 1회 active)
- 데이터 0건이어도 streak 카드 노출(기본 0값)
