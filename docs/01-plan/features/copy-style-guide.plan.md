## 목적

배쓰타임의 카피는 `입욕/샤워 입문자가 오늘 바로 해볼 수 있게 돕는 조용한 가이드`여야 한다.

이 문서는 화면 문구를 쓸 때 아래 판단을 빠르게 내리기 위한 기준이다.

- 지금 이 문구가 입문자에게 바로 이해되는가
- 이 문구가 브랜드 무드보다 실행 장벽을 높이지 않는가
- 이 문구가 같은 앱 안의 다른 화면과 같은 목소리로 들리는가

## 한 줄 원칙

`따뜻하고 차분하게 안내하되, 어렵게 말하지 않는다.`

## 제품 포지션

- 이 앱은 배스 덕후를 위한 전문 도구보다 `입문자가 오늘 한 번 해보게 만드는 앱`에 가깝다.
- 따라서 카피는 깊이보다 실행을 우선한다.
- 준비물도 `없음 또는 완제품 1개` 수준으로 읽혀야 한다.

## 기본 톤

- 친절하다
- 단정하다
- 과장하지 않는다
- 의료적으로 단정하지 않는다
- 럭셔리한 무드는 유지하되 설명은 실용적으로 쓴다

## 카피 레이어

### 1. 브랜드 레이어

브랜드 무드와 감도를 주는 카피.

- 허용: 짧은 영어, 에디토리얼 키워드, 브랜드명
- 역할: 분위기 형성
- 길이: 짧게

예시

- `BATH SOMMELIER`
- `SLEEP EDIT`
- `Quiet luxury`

원칙

- 기능 안내를 대신하면 안 된다
- 버튼, 섹션 제목, 통계 라벨에는 남용하지 않는다

### 2. 가이드 레이어

사용자가 바로 행동하게 만드는 기능 카피.

- 기본 언어: 한국어
- 역할: 이해, 선택, 실행
- 우선순위: 브랜드 무드보다 명확성

예시

- `지금 추천 루틴`
- `준비물 가이드`
- `제품 보기`
- `루틴 시작하기`

원칙

- 처음 보는 사람도 1초 안에 이해해야 한다
- 영어를 섞지 않는다
- 한 문장 안에서 브랜드 톤과 기능 톤을 섞지 않는다

### 3. 안전 레이어

사용을 멈추거나 조심하게 만드는 카피.

- 기본 언어: 한국어
- 역할: 리스크 전달
- 말투: 단호하지만 불안하게 만들지 않음

예시

- `이상하면 바로 중단하세요.`
- `권장 수온과 시간을 먼저 확인해주세요.`

원칙

- 짧게 쓴다
- 회피성 표현보다 행동 지시를 준다
- 의료 효과를 암시하지 않는다

## 언어 규칙

### 한국어 우선

- 기능 라벨, 버튼, 섹션 제목, 통계, 준비물, 안전 안내는 한국어로 통일한다.
- 영어는 브랜드 장식용으로만 제한한다.

### 영어 허용 범위

- 브랜드명
- 제품 에디토리얼 eyebrow
- 아주 짧은 장식성 문구

### 영어 금지 영역

- 버튼
- 주간 통계
- 섹션 제목
- 준비물/안전 안내
- 사용자가 해야 할 행동 설명

## 문장 규칙

### 이렇게 쓴다

- 짧고 부드럽게
- 설명보다 안내 중심으로
- 한 문장에 메시지 하나씩
- 초보자 기준으로 단어를 고른다

예시

- `지금 바로 시작할 수 있어요.`
- `없음 또는 완제품 한 개만 준비하면 돼요.`
- `집에 있는 바디워시로도 충분해요.`

### 이렇게 쓰지 않는다

- 과하게 에디토리얼한 표현
- 실행에 도움이 안 되는 추상어
- 근거 없이 효능을 단정하는 표현
- 한영 혼용

피해야 할 예시

- `정제된 루틴을 큐레이션해드려요`
- `프리미엄 회복 경험`
- `깊은 회복을 보장합니다`
- `Weekly Baths`

## 준비물 카피 규칙

배쓰타임에서 준비물은 `쇼핑 추천`보다 `루틴 실행 보조`로 읽혀야 한다.

### 허용 구조

- `필수 준비물`
- `있으면 좋아요`
- `준비물 보기`

### 권장 메시지

- `새로 사지 않아도 괜찮아요.`
- `집에 있는 바디워시나 샤워타월 정도면 시작할 수 있어요.`
- `이번 루틴 준비물로 보기 좋은 제품만 골라봤어요.`

### 피해야 할 메시지

- `OPTIONAL PAIRING`
- `추가로 얹어보세요`
- `레이어링해보세요`

이유

- 현재 제품 전략은 고급 조합 제안이 아니라 입문자 실행률 상승이기 때문이다.

## 카테고리 카피 규칙

현재 사용자 노출 카테고리는 입문자 기준으로 단순해야 한다.

- `입욕제`
- `샤워 아이템`
- `바디워시`

지금 단계에서는 아래 카테고리를 전면에 내세우지 않는다.

- `에센셜 오일`
- `블렌딩 오일`
- 복수 준비물이 필요한 조합형 표현

## 용어 통일

아래 용어를 기본값으로 사용한다.

- `루틴`
- `준비물`
- `제품`
- `입욕제`
- `샤워 아이템`
- `바디워시`
- `안전 안내`
- `기록`

아래 용어는 특별한 이유가 없으면 피한다.

- `페어링`
- `큐레이션`
- `리추얼`
- `테라피`
- `솔루션`

## 화면별 적용 기준

### 홈

- 브랜드 무드는 허용
- 하지만 핵심 메시지는 `오늘 바로 할 수 있다`로 읽혀야 한다
- 통계와 버튼은 전부 한국어

### 추천/결과 화면

- 가장 실용적인 화면이다
- 브랜드보다 행동 안내가 우선이다
- 준비물, 수온, 시간, 안전은 최대한 직설적으로 쓴다

### 제품 탭

- `에디터 픽`보다 `입문자 추천`에 가깝게 말한다
- 제품을 멋있게 포장하기보다 고르기 쉽게 설명한다

### 안전 모달

- 가장 단정한 톤을 사용한다
- 감성 표현을 넣지 않는다

## 빠른 체크리스트

새 문구를 넣기 전에 아래 5개만 확인한다.

1. 입문자가 바로 이해할 수 있는가
2. 영어 없이도 기능이 전달되는가
3. 이 문구가 쇼핑 유도보다 실행 안내로 읽히는가
4. 브랜드 무드가 기능 이해를 방해하지 않는가
5. 같은 뜻의 말을 다른 화면에서 다르게 부르지 않는가

## 당장 정리할 우선순위

1. 기능 라벨의 영어를 한국어로 통일
2. 준비물/제품 용어를 화면별로 분리
3. 홈과 제품 탭의 에디토리얼 과잉 표현 정리
4. literal string을 `src/content/copy.ts` 중심으로 모으기
