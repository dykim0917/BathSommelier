# Brand Messaging — 바스타임

## Core Shift

### 기존 방향

욕조에서 잘 쉬게 해주는 앱.

### 새 방향

그냥 씻고 끝내던 시간을, 오늘 몸에 맞는 루틴으로 바꿔주는 앱.

이 전환은 바스타임을 "특별한 목욕 경험"보다 "매일 씻는 시간의 작은 변화"에 더 가깝게 둔다. 욕조가 있는 사람만을 위한 앱이 아니라, 샤워만으로도 오늘 몸 상태에 맞는 루틴을 시작할 수 있는 생활형 셀프케어 앱으로 말한다.

## Core Insight

사람들은 보통 "오늘 어떻게 씻지?"라고 고민하지 않는다.
대부분 그냥 씻는다.

바스타임은 이미 존재하는 강한 고민을 해결하기보다, 매일 하는 씻기 행동에 새로운 의미와 방법을 제안한다.

사용자가 떠올려야 하는 장면은 다음과 같다.

> 오늘 좀 피곤하고 찌뿌둥한데, 그때 봤던 것처럼 그렇게 씻어볼까?

## Positioning

바스타임은 피곤한 날, 잠 안 오는 밤, 운동 후 뻐근한 몸처럼 일상적인 컨디션에 맞춰 샤워·목욕 루틴을 가볍게 따라가게 해주는 앱이다.

사용자가 씻는 법을 모른다고 전제하지 않는다. 대신 이미 하던 씻기 시간을 온도, 시간, 순서만 조금 다르게 바꿔보자고 제안한다.

## Representative Message

그냥 씻기 아쉬운 날에

## Supporting Message

피곤한 날, 잠 안 오는 밤, 운동 후 뻐근한 몸까지.
오늘 몸에 맞는 샤워·목욕 루틴을 가볍게 따라가요.

## Messaging Principles

### 해야 할 말

- 그냥 씻던 시간을 조금 다르게 바꿔보자고 제안한다.
- 샤워만으로도 충분히 시작할 수 있다고 말한다.
- 온도, 시간, 순서만 가볍게 확인하면 된다고 말한다.
- 무리하지 않아도 된다고 말한다.
- 오늘 몸 상태에 맞춘 작은 루틴이라고 말한다.
- 루틴을 거창한 성취보다 차분한 실행과 기록으로 보여준다.

### 하지 말아야 할 말

- 사용자가 씻는 법을 모른다는 식으로 말하지 않는다.
- 목욕이나 입욕만 강조하지 않는다.
- 과도하게 힐링, 치유, 치료처럼 말하지 않는다.
- 자기관리 압박처럼 말하지 않는다.
- 너무 전문가처럼 훈계하지 않는다.
- 완료를 대단한 성취처럼 몰아가지 않는다.
- 몽환적이고 추상적인 감성 카피에 기대지 않는다.

## Tone and Manner

- 생활형
- 가볍게
- 차분하게
- 안전하게
- 바로 따라할 수 있게

문장은 짧고 구체적으로 쓴다. 사용자가 앱을 켰을 때 "그래서 지금 뭘 하면 되지?"를 바로 알 수 있어야 한다.

## Copy Set

| Surface | Copy |

| 랜딩 히어로 | 그냥 씻기 아쉬운 날에 |
| 스토어 한 줄 | 매일 씻는 시간을 오늘 몸에 맞는 루틴으로 |
| SNS | 피곤한 날엔 씻는 방식도 조금 다르게 |
| 온보딩 | 샤워만으로도 충분해요 |
| 루틴 상세 | 필요한 것만 보고 바로 시작 |

| 완료 | 오늘의 온도, 시간, 방식을 차분히 남겨요 |

## Message Examples

### Good

- 오늘 몸에 맞는 루틴을 골라보세요.
- 피곤한 날엔 물 온도와 순서만 조금 바꿔봐요.
- 욕조가 없어도 괜찮아요. 샤워만으로 시작할 수 있어요.
- 무리하지 말고, 필요한 단계만 따라가요.
- 씻는 동안은 화면을 오래 보지 않아도 돼요.

### Avoid

- 씻는 법을 알려드릴게요.
- 제대로 쉬려면 입욕이 필요해요.
- 오늘도 나를 관리해야 해요.
- 몸과 마음을 완벽하게 치유하세요.
- 전문가가 설계한 프리미엄 테라피를 경험하세요.

## Product Implications

- 홈 화면은 "목욕하러 가기"보다 "오늘 몸에 맞는 루틴 찾기"를 먼저 보여준다.
- 루틴 추천은 샤워, 족욕, 욕조를 함께 다루되 샤워를 가장 낮은 진입장벽으로 둔다.
- 루틴 상세는 긴 설명보다 온도, 시간, 순서, 주의사항을 먼저 보여준다.
- 타이머는 사용자가 씻는 중에도 부담 없이 따라갈 수 있게 단순하게 유지한다.
- 완료 화면은 성취 배지보다 오늘의 방식과 컨디션을 차분히 남기는 기록에 가깝게 설계한다.
